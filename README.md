# Ioroid's ASS Python Scripts

A collection of Python tools for processing ASS/SSA subtitles.

## Fonts

### [AssFontFamilyNormalize](AssFontFamilyNormalize/README.md)

Normalizes font names in ASS/SSA styles and override tags to canonical font family names.

## Typesetting

### [AssGlyphOpticalCenter](AssGlyphOpticalCenter/README.md)

Horizontally centers subtitle text using its actual visible glyph bounds.
